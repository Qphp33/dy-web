# 位置信息模块抖音风 UI 设计

## 目标

把位置信息模块从偏后台表格风，优化成浅色玻璃态的抖音后台管理界面：毛玻璃、柔和渐变、微光边框、圆润卡片和清晰信息层级。整体参考 Apple HIG 的克制、清晰和层级感，以及 Tailwind UI 常见的浅色玻璃面板、渐变背景和现代卡片布局，同时不改变现有接口、路由和业务逻辑。

## 范围

- 列表页：`src/views/module/Address/index.vue`
- 详情页：`src/views/module/Address/detailPage.vue`
- 不调整接口请求、分页、弹窗、删除、编辑、新建任务等业务行为。
- 不接入新的第三方 UI 库。

## 视觉方向

- 背景：浅蓝到白色的渐变底，叠加低透明度的青色、蓝色、粉色光斑。
- 主色：沿用蓝色 `#1677ff` / `#1890ff`，少量使用抖音红 `#fe2c55` 和青色 `#25d6d2` 做状态和视觉记忆点。
- 容器：半透明白色玻璃面板，使用 `backdrop-filter: blur(...)`、微光渐变边框和柔和阴影。
- 交互：行 hover 和卡片 hover 使用轻微上浮、浅蓝高亮、微光描边和左侧色条。
- 信息层级：标题、任务名、昵称使用深蓝黑；说明、时间、备注使用灰蓝色。

## 页面设计

### 列表页

顶部由普通表格标题改成“位置信息采集”玻璃态任务中心，但保持白蓝浅色：

- 左侧：模块名、说明文案、任务总数、设备总数。
- 右侧：新建按钮和表格设置按钮。
- 表格：保留 Ant Design Vue 表格能力，改为更轻、更圆润的玻璃态浅色表格皮肤。
- 标签、状态、设备 ID：使用胶囊样式，减少后台系统的生硬感。

### 详情页

保留现有结构，重塑视觉：

- 地区卡片保持左图右评论，改成浅色玻璃内容卡，局部使用红青撞色点缀。
- 排序筛选栏保持白色玻璃吸顶工具条，按钮与选择器使用圆角胶囊。
- 用户视频卡片改成玻璃态信息流卡片：头像更突出，属地和 ID 使用浅蓝/浅青胶囊，主要动作仍用蓝色。
- 空状态、分页、弹窗入口保持可用。
